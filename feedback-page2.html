План с экспликацией объекта без обследования

Содержание документа

Содержит технические характеристики объекта на последнюю дату обследования.

Требуется для подготовки проекта перепланировки, проведения оценки или совершения сделки с объектом.

Выдается на бумажном носителе и/или в электронном виде в формате PDF с ЭЦП.

Предоставляется оценщику, проектировщику, Администрацию района/города

Перечень документов для оформления

  • Выписка из ЕГРН либо Акт приема-передачи объекта по ДДУ
